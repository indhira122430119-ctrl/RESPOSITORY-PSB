Sistem Deteksi Pembuluh Vena Manusia

Berbasis Citra Inframerah dan Pemrosesan Digital Terintegrasi

Pengantar Tujuan Implementasi Algoritma Analisis Hasil Akhir

1. Pengantar

Proyek ini mengembangkan sistem deteksi pembuluh vena berbasis citra inframerah untuk membantu prosedur venipuncture secara non-invasif. Sistem ini dirancang untuk meningkatkan akurasi, mengurangi kegagalan tusukan, serta meningkatkan kenyamanan pasien.

2. Tujuan Proyek

  • Mengembangkan prototipe sistem visualisasi vena berbasis NIR
  • Mengimplementasikan pemrosesan citra real-time
  • Menganalisis parameter optimal panjang gelombang & jarak
  • Mendesain casing ergonomis berbasis 3D printing

3. Implementasi Sistem

Diagram alir tahapan implementasi sistem

Sistem menggunakan Raspberry Pi 5 sebagai unit pemrosesan utama, kamera NoIR sebagai sensor citra, serta array LED inframerah (660 nm dan 850 nm) sebagai sumber iluminasi.

Arsitektur sistem perangkat keras

4. Pemrosesan Citra & Algoritma

Pipeline pemrosesan citra digital

Algoritma meliputi konversi grayscale, median blur filter, dan CLAHE untuk meningkatkan kontras lokal vena.

Confusion Matrix hasil pengujian segmentasi

Grafik akurasi dan loss selama pelatihan model

5. Analisis Implementasi

Kurva S pengembangan proyek

Hasil analisis menunjukkan konfigurasi optimal menggunakan panjang gelombang 850 nm pada jarak 30 cm dengan akurasi hingga 95%.

6. Hasil Akhir

Prototipe akhir sistem deteksi vena

Sistem berhasil beroperasi secara stabil, portabel, dan cost-effective. Hasil visualisasi vena menunjukkan peningkatan kontras signifikan pada berbagai karakteristik kulit dan BMI.
